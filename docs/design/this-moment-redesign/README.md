## Extracted design direction

- Mood: warm, lived-in, quiet, and dignified without becoming sentimental or memorial-like.
- Palette: porcelain white, deep garden green, muted persimmon, cornflower blue, honey yellow, and charcoal ink.
- Material: matte paper, linen weave, natural window light, fine rules, and restrained photographic crops.
- Typography: expressive editorial serif headings paired with a readable humanist sans-serif body.
- Layout: open editorial compositions, gapless reading bands, layered image crops, and few purposeful containers.
- Mobile: app-like responsive screens with safe-area spacing, stable bottom navigation, large tap targets, and readable type.
- Crisis state: ordinary navigation is removed and the immediate human-support actions take visual priority.

## Implementation constraint

Generated images can contain imperfect or illustrative body text. Implementation must use the existing tested product copy and the domain boundaries documented in `CONTEXT.md`. In particular, generated explanation text must not introduce diagnoses, prognosis, treatment recommendations, inferred facts, or unsupported crisis claims.
